Advanced P4 trace flags

Optional, off by default. Enables advanced P4 trace diagnostics on the connection used by the plugin, equivalent to the command line flags -vssl, -vrpc, -vnet and -vtime.

Provide a comma separated list of protocol=level pairs, for example:

ssl=2,rpc=3,net=5,time=1

The resulting SSL, RPC and network level output is written to the Jenkins Logger (java.util.logging) under the package org.jenkinsci.plugins.p4, so it can be captured from Manage Jenkins > System Log while reproducing a connection problem and then turned off again by clearing this field. Unrecognised or malformed tokens are ignored.
